# Working in dashboard/

- The **BobNet render-hub** (Nuxt): `npm run dev` on port 3030, `npm run build`. Launch a
  project's team via the hub with `bin/start <uid>` (a thin wrapper on the external launcher).
  The dashboard is optional but recommended.
- The dashboard carries its **own `CLAUDE.md`** (server + UI specifics) — read it when editing here.
- Per-role frontend/dashboard conventions: [`team-rules/tags/dashboard.md`](../../team-rules/tags/dashboard.md).
